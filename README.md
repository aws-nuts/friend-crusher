Friend Crusher is a simple utility to maximize your letters in a single turn of CrossPlay.  It runs in Python using the Django Framework and the MVT (Model-View-Template) architectural pattern.

To run it as a local webserver, you can open a terminal and run the command:

python manage.py runserver

A virtual environment is included utilizing Python 3.13.3 and included dependencies such as Python Regex, Numpy and JSON.

You can run the tests.py scripts to verify installation.
